… の方を好むという表現をする方法

  • prefer を使う

    • S prefer O

      • S は O の方を好む

    • S prefer O to noun

      • S は noun より O の方を好む

    • S prefer to + infinitive 1 rather than to + infinitive 2

      • S は infinitive 2 することよりも infinitive 1 する方を好む

… があるという表現をする方法

  • There is を使う

    • 「… がある」という意味
    • 聞き手にとって新しい情報の時にのみ使える

    • 例文

      • There is an event during class tomorrow.

    • 疑問文

      • Is there any event during class tomorrow?
