  • calm /kάːm/

    • adjective

      • (海・天候など) (波やあらしがなく) 穏やかな、静かな

      • 自信たっぷりの、うぬぼれた

    • transitive verb

    • intransitive verb

      • calm

        • 静まる

    • uncountable noun

    • countable noun

      • 静穏

  • upset /`ʌpsét/

    • transitive verb

      • upset + object

        • object をひっくり返す、ひっくり返して (中身を) こぼす (散乱させる)
        • (計画など) object をくつがえす、だめにする、狂わす
        • (人) object の気を転倒させる、(人) object をろうばい (動転) させる、(人) object を心配させる
        • (人) object の体 (腹) をこわす

      • upset + oneself + about + noun

        • noun のことを心配する、気にする

    • intransitive verb

      • upset

        • (ボートなどが) ひっくり返る、転覆する
        • (容器をひっくり返して) (中身が) こぼれる、散乱する

    • predicative adjective

      • upset + about, by + …

        • … で気が動転して、ろうばいして

      • upset + that + …

        • … ということで気が動転して、うろたえて

    • adjective

      • (胃など) 調子が狂った

    • uncountable noun

      • 転覆、転倒
      • 混乱 (状態)

    • countable noun

      • ろうばい
      • (胃などの) 不調
