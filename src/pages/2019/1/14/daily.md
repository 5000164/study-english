# Words

- bend /bénd/
    - transitive verb
        - bend + object
            - (まっすぐなもの) object を曲げる、(曲がったもの) object を曲げ戻す
            - (写真・封筒など) object を折り曲げる
            - (ひざ) object を折る、(頭部) object を下に向ける、うつむける
    - intransitive verb
        - bend
            - 曲がる、たわむ、しなる
    - countable noun
        - 曲げる (曲がる) こと
        - 曲がり、曲がり目 (角)
