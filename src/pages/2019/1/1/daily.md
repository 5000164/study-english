# 「好きでも嫌いでもない」表現

- I don’t care
    - あんまり興味がない
- It’s OK
    - まあ
        - たぶん、よくはないけど嫌いというわけでもない、というニュアンスだと思う
- I don't like or dislike it
    - 好きでも嫌いでもない

# 「Do you ever」は現在の習慣を聞く

- 「Do you ever + 動詞の原形 … ？」で「普段 … することはありますか？」という意味になる
- 質問の返事には頻度を表す副詞をつけるのが一般的
